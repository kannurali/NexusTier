# Nexus Tier List — редактор

Интерактивный редактор тир-листа по Blox Fruits в стиле макета **Nexus Tier List (сакура)**.
Чистый HTML/CSS/JS, без сборки.

## Как запустить

**Просто дважды кликните `start.bat`** — он поднимет локальный сервер и откроет редактор в браузере
(нужен установленный Python; адрес: http://127.0.0.1:8780/).

> Можно открыть и `index.html` напрямую двойным кликом — редактор работает,
> но кнопка **«Скачать PNG»** при открытии файлом (`file://`) не сработает:
> браузер блокирует экспорт картинок. Поэтому для скачивания используйте `start.bat`.

Альтернатива без `.bat` (в папке проекта):
```
python -m http.server 8780
```
и открыть http://127.0.0.1:8780/

## Возможности

- **Дорожки как в макете**: плашка тира с полутоновыми точками и логотипом по центру,
  дорожка предметов с текстурой, тёмная полоса под ценами, цветные бейджи типов.
- **Тиры**: добавить (`＋ Тир`), переименовать (клик по названию), свой логотип тира (🖼),
  переместить ▲▼, удалить ✕.
- **Предметы**: добавить (`＋` в тире или `＋ Предмет`), перетаскивать между тирами (drag-and-drop).
- **Автосортировка по цене**: при изменении цены предмет сам встаёт на своё место
  (по убыванию, может перейти в другой тир). Выключается тумблером «Автосорт».
  Кнопка «⇅ Сортировать» сортирует все тиры по убыванию цены.
- **Фильтры**: Фрукты / Пермы / Пассы / Скины — можно включать в любых сочетаниях,
  «Все» возвращает всё. Скрытые предметы не попадают в PNG.
- **Редактирование предмета** (клик по ячейке): название, значение, тип (F/S/M/P/GP/CR),
  спрос (🟢🟡🟠🔴), тренд (↑ ⇄ ↓), своя иконка.
- **Логотип в шапке** — фиксированная картинка (флейм БФ + GLH + MK из макета),
  зашит в сайт и не редактируется — знак авторства на каждом экспорте.
- **Реклама** — блок в середине тирлиста, попадает на каждый скрин/скачанное фото.
  Текст редактируется кликом, кнопкой «🖼 Баннер» можно поставить картинку.
- **Команда** — список ролей внизу (Автор, Дизайнер, Аналитик, Помощник аналитика, Кодер сайта).
  Роли и имена редактируются, участников можно добавлять (＋) и убирать (✕).
- **Дата** и тексты редактируются прямо на макете (в режиме редактирования).
- **Режим «Редактирование»** — выключите тумблер, чтобы увидеть чистый вид.
- **Скачать PNG** — сохраняет тир-лист картинкой.
- **Экспорт / Импорт** — сохранить/загрузить весь тир-лист как `.json`.
- **Автосохранение** в браузере (localStorage). **Сброс** — вернуть стандартный шаблон.

## Структура

```
index.html           разметка (тулбар, шапка, тиры, легенда, футер, команда, модалка)
css/styles.css       тема, верстка, дорожки, ячейки
js/app.js            вся логика редактора
js/html2canvas.min.js   экспорт в PNG (локально, офлайн)
assets/              картинки макета + шрифты
  bg.png             фон (сакура/храм/водопад)
  petals.png         лепестки
  dots-band.png      полутоновые точки плашки тира
  dots-tile.png      полутоновая текстура дорожки
  brand-logos.png    логотип шапки (флейм + GLH + MK, из макета)
  logo-flame.png / logo-glh.png / logo-mk.png   логотипы тиров
  bf-logo-trim.png   логотип Blox Fruits (цветной)
  icon-sample.png    стандартная иконка предмета
  fonts/Bootshaus/Bootshaus-Regular.ttf   основной шрифт макета
  fonts/ProtoSans56.otf                   запасной шрифт
```
